Googleスプレッドシート連携 設定手順

1. Googleスプレッドシートを作成
名前例：野田組_安全教育履歴

2. スプレッドシートIDを控える
URLの /d/ と /edit の間の長い文字列です。

3. Google Cloudでサービスアカウント作成
Google Sheets API と Google Drive API を有効化
サービスアカウントのJSONキーを作成

4. JSONの client_email をコピー

5. Googleスプレッドシートを共有
共有 → client_email を追加 → 編集者

6. Streamlit Cloudで公開
GitHubにこのフォルダをアップロード
Main file path は app.py

7. Streamlit Secretsに貼り付け

spreadsheet_id = "スプレッドシートID"

[gcp_service_account]
type = "service_account"
project_id = "JSONのproject_id"
private_key_id = "JSONのprivate_key_id"
private_key = "-----BEGIN PRIVATE KEY-----\n...\n-----END PRIVATE KEY-----\n"
client_email = "JSONのclient_email"
client_id = "JSONのclient_id"
auth_uri = "https://accounts.google.com/o/oauth2/auth"
token_uri = "https://oauth2.googleapis.com/token"
auth_provider_x509_cert_url = "https://www.googleapis.com/oauth2/v1/certs"
client_x509_cert_url = "JSONのclient_x509_cert_url"
